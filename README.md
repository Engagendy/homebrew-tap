# Homebrew Tap

Custom Homebrew tap for [MPP Viewer](https://github.com/Engagendy/MPPViewerMacOS).

## Usage

```bash
brew tap Engagendy/tap
brew install --cask mpp-viewer
```

## Available Casks

| Cask | Description |
|------|-------------|
| `mpp-viewer` | Native macOS viewer for Microsoft Project (.mpp) files |
